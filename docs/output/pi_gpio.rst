###############################
Raspberry Pi GPIO output driver
###############################

This driver works with HD44780-screens connected to Raspberry Pi GPIO. The screen connected has to have its RW pin tied to ground.

Sample config.json:

.. code:: json

    "output":
       [{
         "driver":"pi_gpio",
         "kwargs":
          {
           "pins":[25, 24, 23, 18],
           "en_pin":4,
           "en_pin":17
          }
       }]
